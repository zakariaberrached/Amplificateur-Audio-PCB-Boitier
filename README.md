# Projet Amplificateur Audio – Conception Mécatronique

Ce projet a été réalisé dans le cadre d’un module de conception mécatronique.

## Objectif du projet
L’objectif était de concevoir un système complet d’amplification audio en intégrant les aspects électroniques et mécaniques :
- Conception du circuit électrique d’un amplificateur audio
- Réalisation de la carte PCB à partir du schéma électrique
- Conception d’un boîtier mécanique adapté à la carte électronique

## Travail réalisé

### Partie électronique
- Conception du schéma électrique de l’amplificateur audio
- Réalisation de la carte PCB sous **Altium Designer**

### Partie mécanique
- Conception du boîtier sous **SolidWorks**
- Création d’une pièce inférieure avec trous de fixation pour la carte PCB
- Conception d’un capot supérieur permettant la fermeture du boîtier

## Outils utilisés
- Altium Designer
- SolidWorks
